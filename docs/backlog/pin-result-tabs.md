# Pin Result Tabs

**Tier**: 3 — Nice-to-have
**Type**: frontend
**Inspiration**: DataGrip — Pin result tabs

## Description

Umožnit připnout výsledkový tab, aby nebyl přepsán při spuštění dalšího dotazu. Aktuálně nový dotaz přepíše výsledek v aktivním tabu. Připnutý tab zůstane zachován a nový dotaz otevře nový tab.

### Chování
- Ikona špendlíku na result tabu
- Připnutý tab nelze přepsat automaticky
- Nový dotaz vytvoří nový result tab (pokud je aktuální připnutý)
- Odpřipnutí obnoví normální chování

## Acceptance Criteria

- [ ] Pin ikona na result tabech
- [ ] Připnutý tab není přepsán novým dotazem
- [ ] Nový dotaz vytvoří nový tab když je aktuální připnutý
- [ ] Odpřipnutí funguje kliknutím na pin ikonu
- [ ] Vizuální odlišení připnutého tabu
